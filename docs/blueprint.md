# **App Name**: ShipmentLook

## Core Features:

- Anonymous Authentication: Enable anonymous user authentication upon visiting the page.
- CSV Upload: Upload and process CSV files to update the Firestore database.
- Data Display: Display shipment data in a clear, tabular format.
- Real-time Updates: Display data that updates in real time as the Firestore database is modified.
- Order Search: Allow users to search for shipments using order IDs.
- Status tool Coloring: Color the item background and other UI elements as appropriate for the tracking status
- App Sharing: Enable easy sharing of the app through a shareable link.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a trustworthy and calming feel.
- Background color: Very light Indigo (#F0F2F9) to keep the focus on the data.
- Accent color: Soft Lavender (#C5CAE9) for interactive elements.
- Body font: 'Inter', a sans-serif font known for its readability on screens, ensures clarity and a modern aesthetic for body text.
- Headline font: 'Space Grotesk', a sans-serif font for the headlines.
- Consistent use of 'lucide-react' icons throughout the app for a clean and intuitive user experience.
- A clean, card-based layout is used to organize shipment information, making it easy to scan and digest. This layout ensures key details are immediately accessible and user-friendly.
- Subtle animations are used during data loading and updates to provide feedback and maintain user engagement without being distracting.
